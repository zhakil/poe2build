请实现Path of Building Community for Path of Exile 2计算引擎集成，位于src/poe2build/pob2/:

**目标项目**: PathOfBuildingCommunity/PathOfBuilding-PoE2
- 专门为 Path of Exile 2 设计
- 版本: v0.9.0+ (活跃开发，54 贡献者)
- 功能: PoE2 离线构筑规划器，精确计算系统

1. **calculator.py**: Path of Building Community PoE2 计算接口
   - 构筑数据转换为 Path of Building 导入代码格式
   - Path of Building 进程启动和自动化管理
   - 导入代码自动加载到 Path of Building
   - DPS、防御、生存能力官方级精确计算
   - 计算结果自动导出和解析
   - 错误处理和结果验证

2. **核心计算流程**:
   - 步骤1: 将AI生成的构筑数据转换为PoB2导入代码
   - 步骤2: 启动 Path of Building Community PoE2 客户端
   - 步骤3: 自动导入构筑代码到 Path of Building
   - 步骤4: 触发计算并等待完成
   - 步骤5: 导出计算结果 (XML/JSON格式)
   - 步骤6: 解析结果并标准化输出

3. **支持的计算项目**:
   - 总DPS (主手/副手/法术/持续伤害)
   - 有效生命值 (生命+能量护盾)
   - 抗性计算 (PoE2的80%上限)
   - 暴击率和暴击伤害
   - 攻击/施法速度
   - 精神值消耗验证

功能要求:
- AI提供构筑方案，Path of Building Community PoE2 进行官方级精确计算
- 利用 Path of Building 的完整 PoE2 机制支持 (技能宝石、天赋、装备等)
- 实现与 Path of Building 的完全自动化数据交换
- 实现计算结果的标准化输出
- 包含 Path of Building 不可用时的备用计算方案
