请集成 Path of Building Community for Path of Exile 2 (https://github.com/PathOfBuildingCommunity/PathOfBuilding-PoE2):

**项目验证**: 
- 专门为 Path of Exile 2 设计的离线构筑规划器
- 版本: v0.9.0+ (MIT 许可证，活跃开发中)
- 特性: PoE2 专用伤害/防御计算、天赋树规划、角色导入、构筑分享

**核心原则**: 直接利用现有的 Path of Building Community for PoE2 工具，不重复造轮子

1. **src/poe2build/pob2/path_detector.py**: 
   - Path of Building Community PoE2 安装检测
   - 检测 GitHub 项目: PathOfBuildingCommunity/PathOfBuilding-PoE2
   - 多平台检测 (Windows/Mac/Linux)
   - 常见安装位置: Steam、Epic、独立安装、便携版
   - 可执行文件识别: "Path of Building.exe", "PathOfBuilding", "PoB2.exe" 等
   - Windows注册表检测、进程检测、文件系统搜索
   - 手动路径配置支持

2. **src/poe2build/pob2/local_client.py**:
   - Path of Building Community PoE2 客户端接口
   - 进程管理和启动参数
   - 导入代码自动加载到 Path of Building
   - 计算结果监控和导出
   - 超时和错误处理
   - 版本兼容性检查

3. **src/poe2build/pob2/import_export.py**:
   - 构筑数据 → Path of Building 导入代码转换
   - Path of Building 计算结果解析
   - PoB2 构筑格式标准化
   - 数据验证和清洗

4. **src/poe2build/pob2/calculator.py**:
   - Path of Building Community PoE2 计算接口封装
   - 自动化计算流程管理
   - 结果解析和标准化输出
   - 错误处理和超时管理

实现要求:
- 完全依赖 Path of Building Community PoE2 的官方计算引擎
- 支持多种安装方式的自动检测
- 实现与 Path of Building 的无缝数据交换
- 包含完整的跨平台支持
- 当 Path of Building 不可用时的优雅降级
